# M5 Website Project Documentation

## Overview
This documentation provides an overview of the M5 website project.

## Features
- Responsive design
- User-friendly interface
- Multi-device support

## Installation
To install the project, clone the repository and run the following command:

```
npm install
```

## Usage
To start the development server, use:

```
npm start
```
